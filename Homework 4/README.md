# 📌 PHP Student Management Programs

## 📖 Overview

This project contains multiple **PHP programs** demonstrating core concepts such as file handling, form processing, object-oriented programming (OOP), abstraction, and interfaces.

---

## 📂 Files Included

* `Q1.php` – File handling operations (read, write, append, delete) 
* `file.txt` – Sample text file used for file operations 
* `Q2.php` – Student form with POST method and redirection 
* `thankyou.php` – Confirmation page after form submission 
* `Q3.php` – Student class using constructor and method 
* `Q4.php` – Abstract class with grade calculation 
* `Q5.php` – Interface implementation for course actions 

---

## 🚀 Features

* 📁 File handling (create, read, write, append, delete)
* 📝 HTML form handling with POST method
* 🔁 Page redirection using `header()`
* 🧑‍🎓 Object-Oriented Programming in PHP
* 🧩 Use of abstract classes
* 🔗 Interface implementation

---

## 🛠️ Technologies Used

* PHP
* HTML

---

## 📌 Description

* **Q1:** Demonstrates file operations like reading, writing, appending, and deleting a file
* **Q2:** Takes student input and redirects to a thank you page
* **Q3:** Uses a class to store and display student details
* **Q4:** Uses abstraction to calculate grades for different student types
* **Q5:** Demonstrates interface implementation for course actions
